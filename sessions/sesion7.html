Enunciado del Ejercicio

El objetivo de este ejercicio es que desarrolles un sitio web temático estático utilizando HTML y CSS, con la opción de integrar JavaScript. Tendrás la libertad de elegir cualquier tema que te interese para el sitio. Se permite el uso de Bootstrap, pero el CSS no debe incluir animaciones.

Requisitos del ejercicio:

  • El sitio debe contar con al menos 3 secciones: una página de inicio, una sección temática, y una página adicional.
  • El diseño puede ser responsive, utilizando Bootstrap para ayudar con la disposición y el estilo del contenido. (de manera opcional)
  • Debes incluir al menos un menú de navegación, accesible desde cualquier página.
  • Incorpora al menos 2 imágenes relacionadas con el tema que elijas.
  • El sitio debe tener una estructura clara y coherente, utilizando elementos HTML semánticos.

Detalles adicionales:

  • El uso de JavaScript es opcional, pero se valorará si implementas alguna funcionalidad que aporte interactividad al sitio (por ejemplo: botones o menús dinámicos).
  • El contenido debe estar organizado de forma clara, y la navegación entre páginas debe ser fluida.
  • No está permitido utilizar animaciones en el CSS.

Entrega:

  • **Fecha límite de entrega:** Sábado 26 de octubre.
  • El sitio web debe ser entregado en un repositorio de GitHub o en un archivo comprimido.
  • Asegúrate de que el sitio funcione correctamente antes de entregarlo.
