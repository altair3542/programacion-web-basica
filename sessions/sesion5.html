Objetivo de la clase

Al finalizar esta sesión, los estudiantes serán capaces de utilizar Flexbox y Grid Layout para crear diseños avanzados y responsivos en CSS, organizando de manera flexible el contenido de sus páginas web.

1. Revisión de la Sesión Anterior

2. Introducción a Flexbox

Flexbox es un modelo de diseño unidimensional que facilita la alineación, distribución y ordenamiento de elementos dentro de un contenedor.


.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
}

2.1 Ejes de Flexbox

  • Eje principal (main axis): Dirección principal de los elementos (por defecto, horizontal).
  • Eje transversal (cross axis): Perpendicular al eje principal (por defecto, vertical).

2.2 Propiedades del Contenedor Flex


.contenedor {
  flex-direction: row; /* Los elementos se colocan en fila */
  justify-content: center; /* Centra los elementos en el eje principal */
  align-items: center; /* Centra los elementos en el eje transversal */
}

2.3 Propiedades de los Elementos Flexibles


.elemento {
  flex-grow: 1; /* El elemento crecerá para ocupar el espacio disponible */
  flex-basis: 200px; /* Tamaño inicial del elemento */
}

3. Introducción a CSS Grid Layout

Grid Layout es un modelo bidimensional que permite organizar elementos en filas y columnas.


.grid-contenedor {
  display: grid;
  grid-template-columns: 1fr 2fr; /* Dos columnas, la segunda el doble de la primera */
  grid-template-rows: 100px 200px; /* Dos filas de 100px y 200px */
  gap: 10px; /* Espacio entre los elementos */
}

3.1 Posicionamiento de Elementos en CSS Grid


.elemento {
  grid-column: 1 / 3; /* Ocupa desde la columna 1 hasta la 3 */
  grid-row: 2 / 3; /* Ocupa desde la fila 2 hasta la 3 */
}

5. Flexbox y Grid con Bootstrap

Bootstrap facilita la creación de layouts responsivos con sus utilidades predefinidas de Flexbox y Grid. Puedes consultar más sobre estos sistemas en la documentación oficial de Bootstrap sobre Grid y Flexbox.
