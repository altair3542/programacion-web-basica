2. Introducción a JavaScript, el DOM y TailwindCSS (45 minutos)

JavaScript es el lenguaje de programación que permite la manipulación dinámica del contenido de una página web, proporcionando interactividad mediante la manipulación del DOM (Document Object Model).

2.1 ¿Qué es el DOM?

El DOM es la representación estructurada de los elementos HTML en forma de árbol jerárquico. Con JavaScript podemos acceder a los elementos utilizando selectores como getElementById, querySelector y querySelectorAll.

2.2 Archivos separados para una estructura más limpia

Trabajaremos con HTML, CSS, y JavaScript en archivos separados para mantener una estructura organizada y escalable.

2.3 Insertar y enlazar JavaScript y CSS

En el archivo HTML, enlazamos tanto TailwindCSS como nuestros archivos personalizados de CSS y JavaScript.

3. Eventos en JavaScript con Interacción Dinámica (45 minutos)

Los eventos son acciones que ocurren en una página web cuando el usuario interactúa con ella (clics, escritura, desplazamiento). JavaScript captura estos eventos y ejecuta funciones en respuesta.

3.1 Crear una lista dinámica con eventos

Creamos una lista interactiva que permite agregar elementos, eliminarlos y alternar su visibilidad utilizando JavaScript.

4. Ejercicio Práctico: Crear una Lista Dinámica (45 minutos)

Crearás una página web que incluya un campo para agregar elementos a una lista, un botón para alternar su visibilidad, y otro para eliminar elementos.

4.1 Estructura HTML


<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Lista Dinámica</title>
    <link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
  </head>
  <body class="bg-gray-900 text-white">
    <div class="container mt-5">
      <h1 class="text-2xl mb-4">Lista Dinámica</h1>
      <input type="text" id="itemInput" class="form-control bg-gray-700 text-white p-2 rounded-md" placeholder="Nuevo elemento">
      <button id="addButton" class="bg-blue-500 hover:bg-blue-600 text-white py-2 px-4 mt-2 rounded-md">Agregar</button>
      <button id="toggleButton" class="bg-gray-600 hover:bg-gray-700 text-white py-2 px-4 mt-2 rounded-md">Ocultar Lista</button>
      <ul id="miLista" class="list-none mt-3"></ul>
    </div>
    <script src="script.js"></script>
  </body>
</html>

4.2 Lógica en JavaScript


// Lógica para agregar elementos y alternar la lista
document.addEventListener('DOMContentLoaded', function() {
  const addButton = document.querySelector('#addButton');
  const toggleButton = document.querySelector('#toggleButton');
  const lista = document.querySelector('#miLista');

  addButton.addEventListener('click', function() {
    const input = document.querySelector('#itemInput').value;
    if (input.trim() !== '') {
      const li = document.createElement('li');
      li.textContent = input;
      li.classList.add('bg-gray-800', 'text-white', 'p-2', 'rounded-md', 'mt-2');
      lista.appendChild(li);
    }
  });

  toggleButton.addEventListener('click', function() {
    if (lista.style.display === 'none') {
      lista.style.display = 'block';
      toggleButton.textContent = 'Ocultar Lista';
    } else {
      lista.style.display = 'none';
      toggleButton.textContent = 'Mostrar Lista';
    }
  });
});
