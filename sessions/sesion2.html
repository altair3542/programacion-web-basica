1. Revisión de la Sesión Anterior

Comenzamos revisando los conceptos básicos de HTML aprendidos en la primera sesión, incluyendo etiquetas como <h1>, <p>, <ul>, y <ol>.

2. Formatos de Texto y Estructura en HTML

2.1 Etiquetas de Formato de Texto

Algunas etiquetas importantes para dar formato al texto incluyen:

  • <b> y <strong>: Texto en negrita.
  • <i> y <em>: Texto en cursiva o con énfasis leve.
  • <u>: Subrayar texto (se recomienda usar CSS para este estilo).
  • <small>: Texto de menor tamaño.

2.2 Agrupación de Contenido

Podemos agrupar elementos utilizando:

  • <div>: Agrupa bloques de contenido para darle estilo o layout.
  • <span>: Agrupa contenido en línea.

3. Tablas en HTML

Para organizar datos en filas y columnas:


<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Juan</td>
    <td>30</td>
  </tr>
  <tr>
    <td>Ana</td>
    <td>25</td>
  </tr>
</table>

4. Imágenes en HTML

Para insertar una imagen en una página HTML:


<img src="imagen.jpg" alt="Descripción de la imagen" width="300" height="200">

5. Enlaces en HTML

Crear un enlace de texto:


<a href="https://www.ejemplo.com">Visitar Ejemplo</a>

Crear un enlace de imagen:


<a href="https://www.ejemplo.com">
  <img src="imagen.jpg" alt="Descripción de la imagen">
</a>

6. Ejercicio Práctico

Tarea:

  • Crear una tabla con al menos 3 filas y 3 columnas.
  • Insertar una imagen con texto alternativo y ajustar su tamaño.
  • Añadir al menos dos enlaces: uno interno y uno externo.
